# FreeSWITCH

## Run Dockerfile Locally

1. Build the image locally. (M1 is not supported now: check the issue [here](https://github.com/signalwire/freeswitch/issues/1692))
   ```bash
   make build
   ```
2. Run the image locally.
   ```bash
    make run
   ```
